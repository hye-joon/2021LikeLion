   숙명여대 유일무이 중앙 프로그래밍 동아리 SOLUX


SOLUX는 숙명여자대학교의 유일한 프로그래밍 중앙 동아리입니다.
소프트웨어학부의 리눅스 학회에서 출발한 솔룩스는 이제 더 다양한 분야에서의 도전을 시작합니다.
웹, 아두이노, 빅데이터 등의 분야와 자신의 전공을 융합하고 다양한 분야에 과심 있는 학우들이 모여
IT 프로젝트와 스터디를 진행합니다.
더 나은 자신을 위한 특별한 동아리, 우리는 솔룩스입니다.
솔룩스 티스토리로 이동
2021, 3/26~4/2 1차 세미나
"협업을 위한 Git & Github 이해하기"
Git을 컴퓨터라고 생각한다면, GitHub는 서버다.
Git은 프로젝트의 변경 사항을 추적하고 여러 개발자와 하나의 프로젝트를 동시에 개발할 때, 충돌 문제를 최소화할 수 있도록 도와주는 분산형 버전 관리 시스템 DVCS(Distributed Version Control System)이다. 최근, 전 세계에서 가장 유명한 개발자 커뮤니티인 스택오버플로에서 설문조사를 한 결과 약 90% 이상의 이용자들이 깃 서비스를 이용 중인 것으로 알려졌다. GitHub는 Git을 바탕으로 소스 코드를 호스팅하고, 이를 다른 사람들과 공유하여 협업을 할 수 있도록 도와주는 서비스이다.
2021, 3/28~7/25 기초 스터디
"웹 기초 스터디"
선정된 강의를 지정된 주차까지 들어오고, 그 주 해당 요일에 모여서 질문이나 의문점 등을 공유한다.
1주차부터 5주차까지는 인프런에서 Do it! HTML5 + CSS3 웹 표준의 정석이라는 교재를 통해 공부하고, 6주차부터 9주차까지는 인프런의 WEB2-PHP 강의를 통해 PHP에 대해서 공부한다.
이후 인프런의 생활코딩 mysql, WEB3-PHP&MySQL 강의를 통해 데이터 베이스를 도입하여 배운 내용들을 활용해본다.
